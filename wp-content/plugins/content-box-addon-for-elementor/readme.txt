== Description ==

This content box elementor addon plugin lets you add 7 unique content box addon designs to your website. All element has options to modify its UI like padding, margin, color, typography.

= Features =

* Easy to customize options in Elementor.
* 7 Unique Content Box Designs.
* Align the Content horizontally and much more styles.
* Control the spaces between elements.


== Installation ==

1. Upload the plugin and activate it (alternatively, install through the WP admin console)
2. Edit any page/post using edit with elementor option
3. On elementor widget menu, search/scroll and find the content box addons
4. Drag and drop this addon, where you have to add the section
5. Customize the options and add the content
6. Hurray ! in a minute you have added the Content Box Section
