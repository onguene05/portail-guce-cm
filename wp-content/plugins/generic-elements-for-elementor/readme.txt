Generic Elements is the most complete elementor design toolkit which enhanced the power of elementor plugin.

== Description ==

Generic Elements is an extension of Elementor Page Builder. It has included 14+ custom widgets such as Hero, Slider, PostList, Skill, Team Member, Testimonial, Instagram, etc. There are no limitations for customizations of Generic Elementor Widgets, all the widgets have powerful style controls. There are not included any extra codes that's why it's very Light Weight and super fast loading.

##Key Features

1. No Coding Required
2. Drag and drop widget
3. Fully Responsive
4. Cross-Browser Compatible
5. Made by Elementor
6. Great Support Team
7. Regular Updates
