== Installation ==
1. Upload the Downloaded "image-hover-effects-for-elementor.zip" Folder to the `/wp-content/plugins/` directory
2. Or, UNZIP "image-hover-effects-for-elementor.zip" folder and Upload "image-hover-effects-for-elementor" on "wp-content/plugins/" folder via FTP or Copy and
Paste on your Local Computer
3. Activate the "Image Hover Effects for Elementor" Plugin through the 'Plugins' menu in WordPress
4. Enter the license key you received after the purchase and activate it.
